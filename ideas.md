# DCS Mídia - Brainstorm de Design

## Contexto
Site institucional para agência de marketing político e comunicação estratégica. Objetivo: profissionalizar, passar credibilidade para políticos, gerar autoridade imediata e captar leads.

Identidade visual: **Preto, Branco e Amarelo**
Tom: "Marketing direto ao ponto" - direto, popular, sem jargão.

---

## Resposta 1: Minimalismo Corporativo Ousado
**Probabilidade: 0.08**

### Design Movement
Modernismo corporativo com influência do design suíço - linhas retas, tipografia forte, uso estratégico de espaço negativo.

### Core Principles
1. **Clareza absoluta**: Cada elemento comunica uma função específica
2. **Contraste extremo**: Preto/branco como protagonistas, amarelo como acento cirúrgico
3. **Hierarquia tipográfica**: Diferença radical entre títulos (bold, grande) e corpo (regular, legível)
4. **Respiro visual**: Muito espaço em branco, sem poluição

### Color Philosophy
- **Preto (#000000)**: Autoridade, poder, comando - fundo de seções críticas
- **Branco (#FFFFFF)**: Clareza, confiança, modernidade - espaço respirável
- **Amarelo (#FFD700 ou #FFC107)**: Atenção, urgência, ação - apenas em CTAs e destaques estratégicos
- Intenção: Sensação de "máquina bem oleada", eficiência, profissionalismo sem arrogância

### Layout Paradigm
- Hero com fundo preto, texto branco, CTA amarelo em canto inferior direito
- Seções alternadas: preto/branco, criando ritmo visual
- Grid assimétrico: 60/40 ou 70/30 para quebrar monotonia
- Blocos de conteúdo flutuantes sobre fundo branco

### Signature Elements
1. **Linhas amarelas diagonais**: Separadores de seção, criando movimento
2. **Badges amarelos**: Para destaque de serviços/cases
3. **Tipografia em negrito**: Palavras-chave em amarelo dentro de parágrafos

### Interaction Philosophy
- Hover em amarelo: elementos ganham destaque com transição suave
- Cliques em CTAs: efeito de "pressão" (scale down 2%, depois volta)
- Scroll: seções deslizam com fade-in suave

### Animation
- Fade-in ao scroll: 300ms, ease-out
- Hover em botões: cor amarela mais clara, 150ms
- Separadores diagonais: animação de "desenho" ao entrar na viewport (2s)

### Typography System
- **Display**: Montserrat Bold (700) - 48px/56px para H1
- **Heading**: Montserrat SemiBold (600) - 32px para H2
- **Body**: Inter Regular (400) - 16px/24px
- **Accent**: Palavras-chave em amarelo, sem itálico

## Resposta 2: Brutalismo Digital com Agressividade Política
**Probabilidade: 0.07**

### Design Movement
Brutalismo digital + design de propaganda política - formas geométricas pesadas, tipografia grande e impactante, textura visual.

### Core Principles
1. **Impacto visual imediato**: Nada de subtileza - o design grita
2. **Assimetria radical**: Layouts desalinhados, criando tensão visual
3. **Tipografia como imagem**: Letras gigantes, sobreposições, cortes
4. **Texturas e padrões**: Fundo com ruído, padrões geométricos

### Color Philosophy
- **Preto**: Fundo dominante, sensação de "palco escuro"
- **Branco**: Texto e formas geométricas que "cortam" o preto
- **Amarelo**: Agressivo, chamado à ação, urgência política
- Intenção: Sensação de "campanha em andamento", movimento, conflito positivo

### Layout Paradigm
- Hero com tipografia gigante (80px+) que invade múltiplas linhas
- Imagens cortadas, sobrepostas, criando colagem visual
- Seções com fundo preto com padrão geométrico sutil
- Blocos de texto em colunas desalinhadas

### Signature Elements
1. **Padrão geométrico amarelo**: Triângulos/linhas que "cortam" o layout
2. **Tipografia sobreposta**: Títulos que saem da caixa de conteúdo
3. **Ícones estilizados**: Formas geométricas simples, muito preto e amarelo

### Interaction Philosophy
- Cliques: elementos "explodem" com efeito de partículas
- Hover: fundo amarelo invade o elemento
- Scroll: seções ganham movimento paralaxe

### Animation
- Entrada de elementos: 400ms com efeito de "slide" + rotate
- Hover em cards: amarelo invade de baixo para cima
- Tipografia: palavras-chave ganham outline amarelo ao hover

### Typography System
- **Display**: Bebas Neue (700) - 72px/80px para H1 (impactante)
- **Heading**: Montserrat Bold (700) - 36px para H2
- **Body**: IBM Plex Sans (400) - 16px/24px
- **Accent**: Palavras em amarelo com outline preto

## Resposta 3: Elegância Corporativa com Toque Humano
**Probabilidade: 0.06**

### Design Movement
Design corporativo sofisticado com humanização - linhas suaves, tipografia clássica, fotografia real e autêntica.

### Core Principles
1. **Sofisticação sem frieza**: Corporativo mas com alma
2. **Tipografia clássica**: Serifas em títulos, sans-serif em corpo
3. **Fotografia autêntica**: Pessoas reais, momentos genuínos
4. **Espaçamento generoso**: Respiro visual, elegância através da simplicidade

### Color Philosophy
- **Preto (#1a1a1a)**: Elegância, não frieza - tons ligeiramente quentes
- **Branco (#f9f9f9)**: Sofisticado, não vazio
- **Amarelo (#E8B71A)**: Sofisticado, ouro, luxo - acento refinado
- Intenção: "Agência que entende política E pessoas"

### Layout Paradigm
- Hero com fotografia de fundo (Daniel ou ação política) com overlay preto suave
- Seções com alternância: branco/preto, mas com transições suaves
- Cards com sombra sutil, não planos
- Tipografia em duas colunas para conteúdo longo

### Signature Elements
1. **Linha dourada vertical**: Separador de seções, elegância
2. **Fotografia em moldura**: Cases com bordas pretas, estilo editorial
3. **Citações em amarelo**: Depoimentos com destaque sutil

### Interaction Philosophy
- Hover: elementos ganham sombra mais profunda, não cor
- Cliques: transição suave, sem efeitos bruscos
- Scroll: fade-in suave, movimento natural

### Animation
- Fade-in ao scroll: 500ms, ease-in-out
- Hover em elementos: sombra aumenta, 200ms
- Tipografia: nada de efeitos - elegância através da subtileza

### Typography System
- **Display**: Playfair Display (700) - 52px para H1 (elegância)
- **Heading**: Lora (600) - 28px para H2
- **Body**: Lato (400) - 16px/24px
- **Accent**: Palavras em amarelo ouro, sem efeitos

## Decisão: Resposta 1 - Minimalismo Corporativo Ousado

**Escolhido**: Minimalismo Corporativo Ousado

### Por quê?
1. **Credibilidade política**: Minimalismo é associado a profissionalismo e confiança
2. **Clareza de mensagem**: Sem distrações, o conteúdo é o protagonista
3. **Diferenciação**: Não é "mais um site corporativo" - tem personalidade através da ousadia
4. **Performance**: Design limpo carrega rápido, essencial para conversão
5. **Escalabilidade**: Funciona bem em mobile, tablet e desktop

### Implementação
- **Tipografia**: Montserrat para títulos (força), Inter para corpo (legibilidade)
- **Cores**: Preto (#000000), Branco (#FFFFFF), Amarelo (#FFD700)
- **Espaçamento**: Mínimo 2rem entre seções, 1rem entre elementos
- **Animações**: Fade-in ao scroll, hover suave, transições de 150-300ms
- **Componentes**: Linhas amarelas diagonais como separadores, badges amarelos para destaque
